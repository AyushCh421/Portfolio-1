## About the Project

This is a fully responsive and interactive portfolio designed with a **space-themed UI**.  
The goal was to create a clean and modern interface while keeping performance fast and dependencies minimal.

Instead of relying on heavy UI libraries, I focused on:
- Writing clean React components
- Using CSS Modules for scoped styling
- Implementing custom hooks for animations and interactivity

---

## Features

- ⚡ Smooth scroll animations using Intersection Observer
- 🎯 Dynamic project filtering system
- 🎨 Multiple theme support (Space, Aurora, Solar, Light)
- 💡 Typing animation in Hero section
- 🌌 Custom animated background (Starfield)
- 📱 Fully responsive design

---

## 🛠 Tech Stack

- **Frontend:** React.js (18)
- **Styling:** CSS Modules
- **State Management:** React Hooks
- **Animations:** Pure CSS + custom hooks
- **Deployment:** Vercel
